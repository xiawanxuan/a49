## 1. 产品概述

化学计算实验室分子动力学轨迹可视化系统，为科研人员提供 Web 端高性能的分子运动 3D 动态展示与交互分析工具。

- 核心价值：将分子动力学模拟产生的 XYZ 轨迹数据转化为直观可交互的 3D 可视化，支持原子筛选、视角控制、轨迹参数导出等科研分析功能
- 目标用户：化学、材料、生物物理领域科研人员、学生

## 2. 核心功能

### 2.1 功能模块

1. **主工作台页面**：左侧轨迹文件上传与帧控制栏、中央分子 3D 画布、右侧元素筛选面板

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主工作台 | 轨迹文件上传区 | 支持拖拽/点击上传 .xyz 格式轨迹文件，显示文件信息（帧数、原子数、文件大小） |
| 主工作台 | 帧播放控制栏 | 播放/暂停、上一帧/下一帧、倍速调节（0.25x~4x）、进度条拖拽跳转、帧数显示 |
| 主工作台 | 3D 分子画布 | WebGL 高性能渲染原子球体与化学键，支持 500+ 原子流畅播放 |
| 主工作台 | 视角控制模块 | 拖拽旋转、滚轮缩放、视角保存/加载、自动旋转开关 |
| 主工作台 | 元素筛选面板 | 按元素类型多选高亮/隐藏，原子颜色自定义，一键重置筛选 |
| 主工作台 | 轨迹参数导出 | 导出当前帧坐标、选中原子轨迹、截图 PNG、导出参数为 JSON |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深空灰 #0B1020 背景 + 青色 #00E5FF 科技感主强调色 + 暖橙 #FF8A3D 次强调色
- **辅助色**：原子按元素周期表标准配色（C灰、H白、O红、N蓝、S黄等）
- **按钮风格**：方角扁平、2px 细边框、hover 时发光边框、按压凹陷
- **字体**：JetBrains Mono 等宽字体用于数值显示 + IBM Plex Sans 用于界面文本
- **布局风格**：三栏式固定布局、毛玻璃半透明面板、细边框分割、20px 间距网格
- **图标风格**：Lucide 线性图标，统一 18px 尺寸

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 主工作台 | 左侧控制栏 | 深色半透明面板 320px 宽、文件拖拽区虚线边框、帧进度条、倍速按钮组、时间轴滑块 |
| 主工作台 | 中央3D画布 | 黑色渐变星空背景、居中分子结构、右下角 FPS/帧数 HUD 显示、视角操作提示浮层 |
| 主工作台 | 右侧筛选面板 | 深色半透明面板 280px 宽、元素卡片列表（圆形原子预览+名称+数量）、颜色选择器、开关按钮 |
| 主工作台 | 顶部操作条 | 项目标题、视角保存下拉菜单、导出按钮组 |

### 4.3 响应式设计

- Desktop-first 设计，最低支持 1280px 宽度
- <1280px 时右侧面板自动折叠为抽屉式
- 中央画布自适应容器尺寸

### 4.4 3D 场景指导

- **环境**：深空渐变背景（#050810 → #0B1020 → #111833），无 HDRI
- **光照**：AmbientLight(0.3) + DirectionalLight 主光(强度 1.2, 45° 角度) + 点光源补光
- **相机**：PerspectiveCamera, FOV 50°, 默认距离分子包围球半径 × 2.5
- **交互**：OrbitControls 轨道控制，禁用平移，启用阻尼，旋转速度 0.8，缩放范围 0.3x~10x
- **原子**：SphereGeometry + MeshStandardMaterial（金属度 0.3、粗糙度 0.4），半径按元素范德华半径比例
- **化学键**：CylinderGeometry 圆柱体连接，半径 0.08，根据原子距离动态判定（< 1.6×两原子半径之和）
- **后处理**：轻微 Bloom 发光效果（阈值 0.9，强度 0.4）
- **性能**：InstancedMesh 批量渲染同类原子，几何复用，WebGL2 深度缓冲
